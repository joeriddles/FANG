## Frontend

```shell
cd ./frontend
ng serve --open
```

## Backend

```shell
> cd ./backend
> python -m virtualenv .venv
> ./.venv/Scripts/activate.ps1
(.venv) > pip install -r requirements.txt
(.venv) > cd ..
(.venv) > python -m backend
```
